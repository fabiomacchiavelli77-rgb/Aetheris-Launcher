Fase di Setup

1. Installer — Avvio ed Esecuzione

ID Azione / Test Case Risultato Atteso
1.1 Doppio clic su AristoSeedCrack.bat (Windows) o ./AristoSeedCrack.sh (Linux/Mac) La finestra dell'installer si avvia correttamente, mostrando il logo.
1.2 Eseguire manualmente da riga di comando java -jar Aristois-Donor.jar Stesso risultato del punto 1.1.
1.3 Avviare lo script su un sistema privo di Java installato nel PATH Viene mostrato un messaggio di errore chiaro e descrittivo. Nessun crash silenzioso.

3. Installazione Vanilla + SeedCrackerX

ID Azione / Test Case Risultato Atteso
3.1 Selezionare una versione vanilla, spuntare Install SeedCrackerX e premere Install L'installazione si completa e nella cartella .minecraft/mods/ compare il file Aristois-Seed-Cracker.jar.
3.2 Avviare il gioco con il nuovo profilo Aristois dal Minecraft Launcher Il gioco si avvia correttamente senza crash e con la mod Aristois caricata.
3.3 Digitare in chat: /seedcracker gui Si apre l'interfaccia grafica della mod, divisa in schede di configurazione.
3.4 Verificare la presenza del modulo "Slime Chunks" La scheda "Slime Chunks" è visibile e contiene le nuove opzioni di cracking degli slime.
3.5 Digitare in chat: /seedcracker data e /seedcracker render I comandi vengono riconosciuti, visualizzando lo stato dei dati e attivando/disattivando il rendering dei cuboid.
