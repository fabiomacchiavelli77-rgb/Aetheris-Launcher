Seed Cracker + Anti-Detection Integration
Design Document HTML

Contesto

L'installer Aristois (questo progetto) è un tool desktop che scarica e installa il mod Aristois per Minecraft. Non ha accesso diretto al codice sorgente protetto del mod principale — scarica JAR pre-compilati da maven.aristois.net e da gitlab.com/EMC-Framework/maven.

Obiettivi dell'implementazione:

  1. Seed Cracker: Integrare SeedCrackerX insieme ad Aristois (in modalità Fabric).
  2. Anti-Detection: Rendere i JAR installati più resistenti ai sistemi anti-cheat (es. Grim, Spartan) offuscandone i metadati identificativi.

Dettaglio Tecnico

1. Logica di Patching (JarPatcher.java)

La classe JarPatcher apre il file JAR come un archivio ZIP in memoria ed esegue le seguenti operazioni:

  • Cerca la presenza dei metadati Fabric (fabric.mod.json) o Forge (mcmod.info).
  • Genera un UUID randomizzato a 6 caratteri e lo appende come suffisso all'ID originale della mod (es. aristois_6c9f2b).
  • Modifica la proprietà "name" della mod includendo un suffisso per mimetizzarla (es. Aristois (Optimization)).
  • Scrive i metadati modificati in un file JAR temporaneo ed esegue una sostituzione atomica del file originale per evitare corruzioni in caso di crash.

2. Integrazione UI e Interdipendenze

Nel file SetupScene.java, le opzioni di installazione sono gestite in modo da prevenire conflitti:

// Disabilita SeedCrackerX se l'utente spunta Forge
forgeInstall.withOnChange(() -> {
    if (forgeInstall.isChecked()) {
        seedCracker.setChecked(false);
        seedCracker.setVisible(false);
    } else {
        seedCracker.setVisible(true);
    }
});

3. Meccanismo Offline / Embedded

Per garantire affidabilità e superare i rate limit delle API di GitHub, si è preferito inserire la mod SeedCrackerX direttamente nelle risorse dell'installer (src/main/resources/aristois-seed-cracker.jar). In questo modo l'installazione è garantita in qualsiasi momento, anche offline.
